# Customer Lifetime Value (CLV) Prediction Using Machine Learning

## Introduction

In today's competitive business environment, understanding the value of each customer is essential for making informed marketing and business decisions. Customer Lifetime Value (CLV) is a key metric that estimates the total profit a business can expect to earn from a customer throughout their relationship with the company.

This project presents a Machine Learning-based Customer Lifetime Value Prediction System that analyzes customer acquisition data and predicts the future value of customers. By leveraging advanced regression algorithms, feature engineering techniques, and data visualization, the system helps businesses identify high-value customers, optimize marketing campaigns, and improve customer retention strategies.

The project evaluates multiple Machine Learning models and automatically selects the best-performing model based on prediction accuracy and overall performance.

---

# Project Objectives

The primary objectives of this project are:

* Predict Customer Lifetime Value (CLV) with high accuracy.
* Compare the performance of different Machine Learning regression models.
* Identify the most important factors influencing customer value.
* Segment customers into meaningful business categories.
* Analyze the effectiveness of different marketing channels.
* Generate visual insights that support data-driven decision-making.

# Dataset Description

The project uses a customer acquisition dataset containing information about customer acquisition costs, conversion rates, marketing channels, and generated revenue.

### Dataset Features

| Feature         | Description                                               |
| --------------- | --------------------------------------------------------- |
| Customer ID     | Unique identifier for each customer                       |
| Channel         | Marketing channel through which the customer was acquired |
| Cost            | Acquisition cost spent on obtaining the customer          |
| Conversion Rate | Probability of customer conversion                        |
| Revenue         | Revenue generated by the customer                         |

### Target Variable

Customer Lifetime Value (CLV) is calculated using the following formula:

**CLV = Revenue − Acquisition Cost**

This value represents the net profit generated by a customer and serves as the target variable for prediction.

---

# Feature Engineering

Feature engineering plays a crucial role in improving model performance. Several business-oriented features were created from the original dataset.

### Customer Lifetime Value (CLV)

The CLV feature represents the net value generated by a customer after subtracting acquisition costs from revenue.

### Return on Investment (ROI)

ROI measures the efficiency of marketing spending by comparing generated revenue to acquisition cost.

### Revenue per Conversion Rate

This feature evaluates the amount of revenue generated relative to customer conversion performance.

### Cost Efficiency

Cost efficiency measures how effectively acquisition costs contribute to customer conversion.

These engineered features provide additional information that helps Machine Learning models learn customer behavior more effectively.

---

# Customer Segmentation

To provide deeper business insights, customers are segmented into four categories based on their CLV values using percentile-based grouping.

| Segment  | Description                               |
| -------- | ----------------------------------------- |
| Bronze   | Customers with the lowest lifetime value  |
| Silver   | Customers with moderate lifetime value    |
| Gold     | High-value customers                      |
| Platinum | Customers with the highest lifetime value |

This segmentation helps organizations focus their marketing and retention efforts on the most valuable customer groups.

---

# Machine Learning Models

The project evaluates several regression algorithms to determine the most accurate model for CLV prediction.

### Linear Regression

A simple baseline model that assumes a linear relationship between input features and customer lifetime value.

### Ridge Regression

A regularized version of Linear Regression that helps reduce overfitting and improves model stability.

### Lasso Regression

A regression technique that performs feature selection by reducing the impact of less important variables.

### Random Forest Regressor

An ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and robustness.

### Gradient Boosting Regressor

A powerful boosting algorithm that sequentially improves prediction performance by correcting errors from previous models.

---

# Data Preprocessing

Before training the models, several preprocessing steps were performed.

### Label Encoding

Since marketing channels are categorical values, they were converted into numerical representations using Label Encoding.

Example:

* Email Marketing → 0
* Paid Advertising → 1
* Referral → 2
* Social Media → 3

### Train-Test Split

The dataset was divided into training and testing sets:

* 80% Training Data
* 20% Testing Data

This ensures that the models are evaluated on unseen data and can generalize effectively.

---

# Model Evaluation Metrics

Several evaluation metrics were used to measure model performance.

### R² Score

Measures how well the model explains variations in customer lifetime value.

### Mean Absolute Error (MAE)

Represents the average difference between actual and predicted CLV values.

### Root Mean Squared Error (RMSE)

Measures prediction error while giving higher importance to larger mistakes.

### 5-Fold Cross Validation

Used to evaluate model stability and ensure reliable performance across different subsets of data.

# Results and Performance

The dataset contained:

* 800 customer records
* 11 total features
* 640 training samples
* 160 testing samples

### CLV Statistics

* Average CLV: $2756.00
* Standard Deviation: $1259.40
* Minimum CLV: $479.55
* Maximum CLV: $4985.68

After comparing all models, the **Gradient Boosting Regressor** achieved the highest performance.

### Best Model Performance

| Metric                  | Value  |
| ----------------------- | ------ |
| Cross Validation R²     | 0.9989 |
| Test R²                 | 0.9994 |
| Mean Absolute Error     | $24.23 |
| Root Mean Squared Error | $32.19 |

The results demonstrate excellent predictive accuracy and strong generalization capability.
